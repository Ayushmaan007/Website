Install Kali Linux on Virtual Box (ver2020.3)

For installing Kali linux in Virtual Box First we would be needing some prerequisites.
1.Kali Linux Image file from their official page KALI
2.Virtualization software here we are having Virtual Box.
3.Internet connection as kali linux comes with preloaded tools but there are some things we would be downloading.
First we have to understand, What is virtual box and why are we installing kali linux on virtual box ?
Virtual box is a Virtualization software simply after installing it we can call it a computer in a computer, it behaves like a computer and work like a computer but with a twist we can control it fully like to give access to network adapter, Ram and no. of cpus to use, and many other things like this like when we create a virtual machine it creates an .vdi file its same as a hardisk in a computer where OS files and other files are stored and this .vdi extension is only recognised by virtual box so this means after creating a virtual machine the main hardisk which is the main computer where virtual box is installed is not affected
